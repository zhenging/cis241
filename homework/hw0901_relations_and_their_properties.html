25. How many different relations are there from a set with mmm elements to a set with nnn elements?

Solution

Let the set with mmm elements be AAA, and the set with nnn elements be BBB.
1. A relation from AAA to BBB is a subset of A×BA \times BA×B.
2. By product rule, A×BA\times BA×B has mnmnmn elements, so it has 2mn2^{mn}2mn subsets.
3. Thus there are 2mn2^{mn}2mn relations from AAA to BBB.
