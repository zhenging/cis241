Definition 1

Let AAA and BBB be sets. A binary relation RRR from AAA to BBB is a subset of A×BA \times BA×B.

A×B={(a,b)∣a∈A∧b∈B}R⊆A×B\begin{aligned} A \times B &= \{(a, b) \mid a\in A \land b \in B\}\\ R &\subseteq A \times B \end{aligned} A×BR​={(a,b)∣a∈A∧b∈B}⊆A×B​

Example 1
Let AAA be the set of students in your school, and let BBB be the set of courses. Let RRR be the relation that consists of those pairs (a,b)(a, b)(a,b), where aaa is a student enrolled in course bbb. For instance, if Jason Goodfriend and Deborah Sherman are enrolled in CS518, the pairs (Jason Goodfriend, CS518) and (Deborah Sherman, CS518) belong to RRR. If Jason Goodfriend is also enrolled in CS510, then the pair (Jason Goodfriend, CS510) is also in RRR. However, if Deborah Sherman is not enrolled in CS510, then the pair (Deborah Sherman, CS510) is not in RRR.
Note that if a student is not currently enrolled in any courses there will be no pairs in RRR that have this student as the first element. Similarly, if a course is not currently being offered there
will be no pairs in RRR that have this course as their second element.

Example 2
Let AAA be the set of cities in the U.S.A., and let BBB be the set of the 50 states in the U.S.A. Define the relation RRR by specifying that (a,b)(a, b)(a,b) belongs to RRR if a city with name aaa is in the state bbb. For instance, (Boulder, Colorado), (Bangor, Maine), (Ann Arbor, Michigan), (Middletown, New Jersey), (Middletown, New York), (Cupertino, California), and (Red Bank, New Jersey) are in RRR.

EXAMPLE 12
Is the “divides” relation on the set of positive integers symmetric? Is it antisymmetric?
Solution: This relation is not symmetric because 1∣21 \mid 21∣2, but 2∤12 \nmid 12∤1. It is antisymmetric, for if a and b are positive integers with a∣ba \mid ba∣b and b∣ab \mid ab∣a, then a = b (the verification of this is left as an exercise for the reader).

EXAMPLE 15
Is the “divides” relation on the set of positive integers transitive?
Solution: Suppose that aaa divides bbb and bdividescb divides cbdividesc. Then there are positive integers kkk and lll such that b=akb = akb=ak and c=blc = blc=bl. Hence, c=a(kl)c = a(kl)c=a(kl), so aaa divides ccc. It follows that this relation is transitive.

EXAMPLE 16
How many reflexive relations are there on a set with n\mathsf{n}n elements?
Solution: A relation RRR on a set AAA is a subset of A×AA\times AA×A. Consequently, a relation is determined by specifying whether each of the n2n^2n2 ordered pairs in A×AA \times AA×A is in RRR. However, if RRR is reflexive, each of the nnn ordered paries (a,a)(a, a)(a,a) for a∈Aa \in Aa∈A must be in RRR. Each of the other n(n−1)n(n-1)n(n−1) ordered pairs of the form (a,b)(a, b)(a,b), where a≠ba\ne ba≠b, may or may not be in RRR. Hence, by the product rule form counting, there are 2n(n−1)2^{n(n-1)}2n(n−1) reflexive relations [this is the number of ways to choose whether each element (a, b), with a≠ba \ne ba≠b, belongs to RRR].

Definition

which there exists an element b∈Bb \in Bb∈B such that (a,b)∈R(a, b) \in R(a,b)∈R and (b,c)∈S(b, c) \in S(b,c)∈S. We denote the composite of R and S by S∘RS \circ RS∘R.
