1.1 Propositional Logic

  • Proposition - A statement that has a truth value.
  • Truth table and how to construct one.
  • Logical operations - and (∧\land∧), or (∨\lor∨), negation (¬\neg¬).
  • Implication (If assumption is false, the conclusion is always true).

    p→qp\to qp→q, if ppp, then qqq.
    If ppp (assumption) is false, p→qp\to qp→q (conclusion) is always true.

  • Bi-implication (Double implication, if and only if) p⟺q≡(p→q∧q→p)p \iff q \equiv (p \to q \land q \to p)p⟺q≡(p→q∧q→p).
  • Contrapositive - The contrapositive of p→q is ¬q→¬pp\to q \text{ is } \lnot q \to \lnot pp→q is ¬q→¬p.
