Let ppp be the statement (i) 3x+23x + 23x+2 is even, qqq be the statement (ii) x+5x + 5x+5 is odd, and rrr be the statement (iii) x2x^2x2 is even. In order to show that these statement are equivalent about the integer xxx, we need to show (1)p→qp \to qp→q, (2)q→rq \to rq→r, and (3) r→pr \to pr→p.

x+5=2kx=2k−53x+2=3(2k−5)+2=2(3k−7)+1\begin{aligned} x + 5 &= 2k \\ x &= 2k-5 \\ 3x+2 &= 3(2k-5) + 2 = 2(3k-7) + 1 \end{aligned} x+5x3x+2​=2k=2k−5=3(2k−5)+2=2(3k−7)+1​

Therefore, 3x+23x+23x+2 is odd. p→qp \to qp→q is proved.
(2) q→rq \to rq→r. If (ii) x+5x + 5x+5 is odd, then (iii) x2x^2x2 is even.
Let x+5=2k+1x + 5 = 2k + 1x+5=2k+1, where kkk is an integer.

x+5=2k+1x=2k−4x2=(2k−4)2=2(2k2−8k+8)\begin{aligned} x+5 &= 2k+1\\ x &= 2k-4\\ x^2 &= (2k-4)^2 = 2(2k^2 - 8k + 8) \end{aligned} x+5xx2​=2k+1=2k−4=(2k−4)2=2(2k2−8k+8)​

(3) r→pr \to pr→p. If (iii) x2x^2x2 is even, then (i) 3x+23x + 23x+2 is even.
We first need to proof that if x2x^2x2 is even, then xxx is even. By contraposition, the contrapositive is "if xxx is odd, then x2x^2x2 is odd". Let x=2k+1x = 2k + 1x=2k+1, where kkk is an integer.

3x+2=3×2m+2=2(3m+1)3x + 2 = 3 \times 2m + 2 = 2(3m+1) 3x+2=3×2m+2=2(3m+1)
