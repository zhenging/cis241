11.1 Introduction To Trees

A tree is a connected undirected graph with no simple circuits.

Theorem 1

An undirected graph is a tree if and only if there is a unique simple path between any two of its vertices.

Suppose that TTT is a rooted tree. If vvv is a vertex in TTT other than the root, the parent of vvv is the unique vertiex uuu such that there is a direct edge from uuu to vvv. When uuu is the parent of vvv, vvv is called a child of uuu. Vertices with the same parent are called siblings. The ancestors of a vertex other that the root are the vertices in the path from the root to this vertex, excluding the vertex itself and including the root (that is, its parent, its parent's parent, and so on, until the root is reached). The descendants of a vertex vvv are those vertices that have vvv as an ancestors. A vertex of a rooted tree is called a leaf if it has no children. Vertices that have children are called internal vertices. The root is an internal vertex unless it is the only vertex in the graph, in which case it is a leaf.
If aaa is a vertex in a tree, the subtree with aaa as its root is the subgraph of the tree consisting of aaa and its descendants and all edges incident to these descendants.

Theorem 2

A tree with nnn vertices has n−1n-1n−1 edges. (Proof by induction. The extra vertex must have a degree of 1, otherwise the graph would contain a cycle and would not be a valid tree.)

Theorem 3

A full m-ary tree with iii internal vertices contains n=mi+1n = mi + 1n=mi+1 vertices.

28. A complete m-ary tree is a full m-ary tree in which every leaf is at the same level. How many vertices and how many leaves does a complete m-ary tree of height h have?

Solution

A complete m-ary tree of height h has m0+m1+⋅+mhm^0 + m^1 + \cdot +m^hm0+m1+⋅+mh vertices and mhm^hmh leaves.
Proof by induction.
Basic step When h = 0, the tree has m0=1m^0 = 1m0=1 vertex and m0=1m^0 = 1m0=1 leaves.
Hypothesis Assume that the number of leaves for a complete m-ary tree with height k is mkm^kmk and the number of vertices is m0+m1+⋯+mkm^0 + m^1 + \cdots +m^km0+m1+⋯+mk.
Inductive Step We need to prove that the number of leaves for a complete m-ary tree with height k+1k+1k+1 is mk+1m^{k+1}mk+1 and the number of vertices is m0+m1+⋯+mk+mk+1m^0 + m^1 + \cdots +m^k + m^{k+1}m0+m1+⋯+mk+mk+1.
Let TTT be a complete m-ary tree of height k+1k+1k+1 and consider a subgraph T′T'T′ that results from removing all leaves from TTT. T′T'T′ is a complete m-ary tree of height kkk, so T′T'T′ has mkm^kmk leaves and m0+m1+⋯+mkm^0 + m^1 + \cdots +m^km0+m1+⋯+mk vertices (By hypothesis).
Each leaf of T′T'T′ has mmm children in TTT and each leaves of TTT is the child of some leaf of T′T'T′. Therefore, TTT has m⋅mk=mk+1m \cdot m^k = m^{k+1}m⋅mk=mk+1 leaves and m0+m1+⋯+mk+mk+1m^0 + m^1 + \cdots +m^k + m^{k+1}m0+m1+⋯+mk+mk+1 vertices.
