11.3 Tree Traversal

Preorder

Let TTT be an ordered rooted tree with root rrr. It TTT consists only of rrr, then rrr is the preorder trasversal of TTT. Otherwise, suppose that T1,T2,⋯TnT_1, T_2, \cdots T_nT1​,T2​,⋯Tn​ are the subtrees at rrr from left to right in TTT. The preorder trasveral begins by visiting T1T_1T1​. It continues by traversing T1T_1T1​ in preorder, then T2T_2T2​ in preorder, and so on, until TnT_nTn​ is traversed in preorder.

Root => Left => Right

Algorithem 1 Preorder traversal
procedure preorder(T: ordered rooted tree)
r := root of T
list r
for each child c of r from left to right
  T(c) := subtree with c as its root
  procedure(T(c))

Inorder

Let TTT be an ordered rooted tree with root rrr. It TTT consists only of rrr, then rrr is the inorder trasversal of TTT. Otherwise, suppose that T1,T2,⋯TnT_1, T_2, \cdots T_nT1​,T2​,⋯Tn​ are the subtrees at rrr from left to right in TTT. The inorder trasveral begins by traversing T1T_1T1​ in inorder, then visiting rrr. It continues by traversing T2T_2T2​ in inorder, then T3T_3T3​ in inorder, ..., and finally TnT_nTn​ in order.

Left => Root => Right

Algorithem 2 Inorder traversal
procedure inorder(T: ordered rooted tree)
r := root of T
if r is a leaf then list r
else
  l := first child of r from left to right
  T(l) := subtree with l as its root
  inorder(T(l))
  list r
  for each child c of r except l from left to right
    T(c) := subtree with c as its root
    inorder(T(c))

Postorder

Let TTT be an ordered rooted tree with root rrr. It TTT consists only of rrr, then rrr is the postorder trasversal of TTT. Otherwise, suppose that T1,T2,⋯TnT_1, T_2, \cdots T_nT1​,T2​,⋯Tn​ are the subtrees at rrr from left to right in TTT. The postorder trasveral begins by traversing T1T_1T1​ in postorder, then T2T_2T2​ in postorder, ... then TnT_nTn​ in postorder, and ends by visiting rrr.

Left => Right => Root

Algorithem 3 Postorder traversal
procedure postorder(T: ordered rooted tree)
r := root of T
for each child c of r from left to right
  T(c) := subtree with c as its root
  postorder(T(c))
list r
